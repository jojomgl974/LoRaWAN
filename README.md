# Utilisation de LoRaWAN avec Module Dragino LA66
Utilisation du Dragino LA66 avec Arduino Mega

# Matériel
- Raspberry Pi
- Passerelle RAK7246G
- Module Draguino LA66
- Ordinateur
- Arduino Mega

# Installation du matériel
- Emboiter le Dragino LA66 à la Raspberry Pi
- Installer la passerelle en suivant ce tutoriel
→ https://docs.rakwireless.com/product-categories/wisgate/rak7246g/quickstart/
- Créer un compte TTN pour mettre en place les serveur réseau et application LoRaWAN
- Créer une interface web qui communiqaue avec le serveur TTN

# Déroulement de l'expérience
- Programmer l'envoi d'un uplink depuis l'interface web
- Programmer l'envoi d'un downlink depuis l'interface web
- Programme le contrôle d'une LED depuis l'interface web
